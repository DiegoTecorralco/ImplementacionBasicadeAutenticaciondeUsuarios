# Implementación Básica de Autenticación de Usuarios
Actividad práctica guiada para comprender cómo funciona un sistema de autenticación web, aplicando buenas prácticas en el manejo de credenciales, roles y niveles de acceso.

## Objetivo de la Práctica

El estudiante implementará un sistema básico de autenticación en una aplicación web, comprendiendo la relación entre usuarios, roles y niveles de acceso, y aplicando buenas prácticas en el manejo de credenciales desde el primer momento.

- **Comprender:** El flujo de autenticación y sus componentes esenciales en una app web.
- **Implementar:** Un sistema de login funcional con HTML y JavaScript desde cero.
- **Aplicar:** Buenas prácticas de seguridad: validaciones, roles y manejo de contraseñas.

## ¿Qué vamos a construir?
**El escenario**
Se desarrollará un sistema sencillo de inicio de sesión para una aplicación web, simulando un sistema institucional o una tienda en línea. Los usuarios deberán autenticarse para acceder a funcionalidades según su rol asignado.
